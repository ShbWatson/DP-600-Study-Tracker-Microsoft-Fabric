# DP-600 Study Tracker (Microsoft Fabric Analytics Engineer)

This repo contains the resources I used to track my progress while preparing for Exam DP-600: Implementing Analytics Solutions Using Microsoft Fabric.

## Important note

The Excel file in this repo is based on the official DP-600 study guide and is designed to dynamically pull the skills measured from that page.

However, Microsoft may change the structure or content of the study guide at any time. If that happens, the Excel file may not refresh correctly or may not reflect the latest skills.

For that reason, you should always refer to the official study guide as the source of truth and use this tracker as a companion tool.

## What’s included

This repo contains the following resources:

1. **Excel tracker**  
   An Excel file based on the official study guide. It is designed to pull and organize the skills measured so you can track your progress.

2. **Power BI dashboard**  
   A Power BI file built on top of the Excel tracker to visualize progress and identify areas that need more focus.

## How to use

The Excel file and Power BI file work together:

1. Download both files.
2. Open the Power BI file (`DP600-StudyTracker.pbix`).
3. Go to:
   - Transform Data → Data source settings
4. Update the Excel file path to point to where you saved the Excel file (`DP600-SkillsMeasured-Tracker.xlsx`).
5. Apply changes.

Tip: Keep both files in the same folder to make this step easier.

## My study approach

I used the following approach while preparing:

- Break the exam down into skills measured  
- Track progress for each skill in Excel  
- Use Power BI to monitor overall progress  
- Focus on areas with lower confidence  
- Do a full review before the exam
